# SoftLicence Branding Assets

Ce dossier contient les ressources graphiques officielles du projet.

- `icon.svg` : L'icône bouclier (utile pour les boutons ou la barre de titre WPF).

**Couleur primaire** : `#0d6efd` (Bleu Bootstrap/Industrial)
**Thème** : Dark Mode par défaut.
